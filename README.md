# Atividade-servo

Este projeto foi desenvolvido na plataforma Wokwi utilizando 2 push buttons, 2 LEDs, 2 resistores, 1 servo motor e 1 display OLED SSD1306.

O funcionamento ocorre da seguinte forma:

**Push Button 1 (azul):** ao ser pressionado, o servo motor gira para a posição de 90°, o LED azul é aceso e o display OLED SSD1306 exibe a mensagem **"Luz acesa"**.

**Push Button 2 (verde):** ao ser pressionado, o servo motor retorna para a posição inicial de 0°, o LED verde é aceso e o display OLED SSD1306 exibe a mensagem **"Luz apagada"**.

Dessa forma, o sistema simula o acionamento e o desligamento de uma luz, fornecendo uma indicação visual por meio dos LEDs e uma confirmação textual no display.
